# Customer Segmentation using K-Means Clustering  

## 1. Overview  
Customer segmentation is a technique used to divide customers into groups based on similar characteristics. This project applies the K-Means Clustering algorithm to identify different customer segments and help businesses make data-driven marketing decisions.

---

## 2. Objectives  
- Group customers based on similar features  
- Identify high-value and low-value customers  
- Improve marketing strategies  
- Increase business efficiency and customer satisfaction  

---

## 3. Technologies Used  
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  
- Google Colab  

---

## 4. Dataset Features  
- Customer ID  
- Gender  
- Age  
- Annual Income (k$)  
- Spending Score (1–100)  
- Purchase Frequency  

---

## 5. Methodology  

### 5.1 Data Collection  
- Collected customer dataset  

### 5.2 Data Preprocessing  
- Cleaned data  
- Handled missing values  
- Prepared features

- Analyzed distributions  
- Visualized relationships between features

- Selected relevant variables for clustering

- Determined optimal number of clusters (K)

- Applied clustering algorithm

### 5.7 Silhouette Score Evaluation  
- Evaluated clustering quality  

### 5.8 Cluster Visualization  
- Plotted clusters using scatter plots  

### 5.9 Cluster Interpretation  
- Interpreted customer groups

- Compared results with DBSCAN  

---

## 6. Results  
- Customers are divided into 5 clusters  
- Clear segmentation based on income and spending patterns  
- Visual insights using graphs and plots  

---

## 7. Cluster Insights  
- Premium Customers → High income, high spending  
- Budget Customers → Low income, low spending  
- Target Customers → High income, low spending  
- Risk Customers → Low income, high spending  
- Average Customers → Medium values  

---

## 8. Business Insights  
- Premium customers can be targeted with loyalty programs  
- Budget customers can be attracted with discounts  
- Target customers are ideal for focused marketing campaigns  
- Personalized strategies improve customer engagement  

---

## 9. Algorithm Comparison  
- K-Means → Best for structured data and clear clusters  
- DBSCAN → Useful for detecting noise and outliers

# Customer Segmentation using K-Means Clustering

## 📌 Project Overview

Customer segmentation is a technique used to divide customers into groups based on similar characteristics.
This project uses the **K-Means Clustering Algorithm** to identify different types of customers and help businesses make better marketing decisions.

---

## 🎯 Objectives

* Group customers based on similar features
* Identify high-value and low-value customers
* Improve marketing strategies
* Increase business efficiency and customer satisfaction

---

## 🧠 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Google Colab

---

## 📊 Dataset Features

* Customer ID
* Gender
* Age
* Annual Income (k$)
* Spending Score (1–100)
* Purchase Frequency

---

## ⚙️ Methodology

1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. Elbow Method (to find optimal K)
6. K-Means Clustering
7. Silhouette Score Evaluation
8. Cluster Visualization
9. Cluster Interpretation
10. Comparison using DBSCAN

---

## 📈 Results

* Customers are divided into 5 clusters
* Clear segmentation based on income and spending
* Visual insights using scatter plots and graphs

---

## 🧩 Cluster Insights

* Premium Customers → High income, high spending
* Budget Customers → Low income, low spending
* Target Customers → High income, low spending
* Risk Customers → Low income, high spending
* Average Customers → Medium values

---

## 💡 Business Insights

* Premium customers can be targeted with loyalty programs
* Budget customers can be attracted with discounts
* Target customers are ideal for marketing campaigns
* Personalized strategies improve customer engagement

---

## 🔄 Algorithm Comparison

* K-Means → Best for structured data and clear clusters
* DBSCAN → Identifies noise and outliers
